
# SOC Portfolio (Entry-Level)

Egresado de Ingeniería en Informática (Santiago, Chile).

Construir evidencia práctica para postular a roles Junior.

## Contenido
- `notes/`: apuntes prácticos (networking, web, Linux, Windows, logs, SOC)
